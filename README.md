# Shepherd

Personal developer command center with gamification. Earn XP and level up for pushing commits and solving LeetCode problems. Track GitHub and LeetCode streaks, manage your job search with a Kanban board, and get weekly AI digests — all in one place.

## Stack

- **Frontend** — Next.js, Tailwind CSS, shadcn/ui
- **Backend** — Python FastAPI, PostgreSQL, Redis
- **Auth** — GitHub OAuth + JWT
- **AI** — Anthropic Claude, OpenAI
